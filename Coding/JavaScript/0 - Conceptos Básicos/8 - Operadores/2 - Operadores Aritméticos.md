En JavaScript, los operadores aritméticos son utilizados para realizar cálculos matemáticos.
```javascript
let num1 = 10;
let num2 = 2;

// Operaciones aritméticas
let suma = num1 + num2;              // Suma
let resta = num1 - num2;             // Resta
let multiplicacion = num1 * num2;    // Multiplicación
let division = num1 / num2;           // División
let modulo = num1 % num2;             // Módulo
let exponenciacion = num1 ** num2;    // Exponenciación

// Incremento y Decremento
let c = 5;
c++;                                  // Incremento (c ahora es 6)

let d = 5;
d--;                                  // Decremento (d ahora es 4)

// Resultados
console.log(`Suma: ${suma}`);                 
console.log(`Resta: ${resta}`);               
console.log(`Multiplicación: ${multiplicacion}`); 
console.log(`División: ${division}`);           
console.log(`Módulo: ${modulo}`);               
console.log(`Exponenciación: ${exponenciacion}`); 
console.log(`Valor de c (incrementado): ${c}`); 
console.log(`Valor de d (decrementado): ${d}`);
```
